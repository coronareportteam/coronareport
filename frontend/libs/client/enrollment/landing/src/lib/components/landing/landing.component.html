Herzlich Willkommen bei quarano, dem Online-Dokumentationstool Ihrer Gesundheitsbehörde!

Mit dem persönlichen Registrierungs-Code bzw. dem personalisierten Link melden Sie sich nun bitte auf den folgenden Seiten an und nutzen dieses Tool als online Dokumentation*.

Dazu werden Sie im ersten Schritt aufgefordert, sich mit einem Benutzernamen und einem Passwort zu registrieren. Im darauf folgenden Schritt werden ein paar notwendige Basisdaten abgefragt, bevor Sie dann die sogenannten „direkten Kontaktpersonen” angeben können, zu denen Sie seit Symptombeginn (und bis zu 48h zuvor) Kontakt hatten.

Diese Informationen müssen Sie nur einmal initial eingeben. Danach können Sie direkt mit dem ersten Symptomtagebucheintrag beginnen. Hier tragen Sie möglicherweise auftretenden Symptome tagesaktuell ein (wie z. B. Fieber, Atembeschwerden, Glieder- oder Muskelschmerzen, Durchfall, …)

Diese Informationen sind von Ihnen ab sofort und während der gesamten Quarantäne / der häuslichen Isolation 2x am Tag (morgens und abends) einzugeben.

Wichtig: Wenn mehrere betroffene Personen im selben Haushalt leben, ist es wichtig, dass sich jede Person mit einem eigenen Code / Zugangslink vom Gesundheitsamt jeweils einen eigenen Benutzer anlegt. So können alle Einträge immer richtig zugeordnet werden.

Vielen Dank, dass Sie uns mit der Selbsteingabe in unser Online-Dokumentationstool quarano unterstützen, der dynamischen Entwicklung der Fallzahlen gerecht zu werden.
